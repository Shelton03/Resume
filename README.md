# Shelton Mutambirwa — Terminal Portfolio

A multi-page, terminal-inspired portfolio site built with static HTML/CSS. Optimized for Vercel static deployment and focused on backend/SRE storytelling.

## Pages

- `index.html` — home / mission control
- `about.html` — background and education
- `experience.html` — professional roles
- `projects.html` — featured builds + open source highlights
- `skills.html` — skills matrix + certifications
- `contact.html` — contact info and links

## Local preview

Open `index.html` in your browser or use any static file server.

## Deployment

This is a static site and works with Vercel's default static hosting. No build step required.
